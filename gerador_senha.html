Gerador de Senha

O Gerador de Senha é uma ferramenta online que permite aos usuários criar senhas seguras e personalizadas. Com opções para incluir letras maiúsculas, minúsculas, números e caracteres especiais. Criado para treinar habilidades em JavaScript, HTML e CSS, focando em usabilidade, design responsivo, manipulação de DOM e lógica de programação.

Tecnologias utilizadas

  • HTML
  • CSS
  • JavaScript
