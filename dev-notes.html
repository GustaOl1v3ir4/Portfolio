Dev Notes

O Dev Notes é uma ferramenta um sistema de anotações interativas utilizando HTML, CSS e JavaScript puro. A aplicação permite criar, editar, fixar, duplicar e excluir notas, com persistência automática via Local Storage. Inclui busca em tempo real e exportação de todas as anotações em formato CSV.

Tecnologias utilizadas

  • HTML
  • CSS
  • JavaScript
