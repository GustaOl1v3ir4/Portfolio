Quiz de Programação

O Quiz de programação é um projeto realizado em React criado com o objetivo de praticar gerenciamento de estado global utilizando Context API. A aplicação possui um fluxo completo de perguntas e respostas, controle de progresso, pontuação e transições entre telas, tudo centralizado em um único estado global. Isso elimina a necessidade de “prop drilling” e deixa o código mais organizado e escalável — simulando a arquitetura de aplicações reais que utilizam Redux, mas de forma nativa com React.

Tecnologias utilizadas

  • HTML
  • CSS
  • JavaScript
  • React
