Dashboard Financeiro

O Dashboard Financeiro é um painel de visualização de dados que apresenta métricas financeiras como receitas, despesas e lucros mensais. Ele foi desenvolvido com foco em design limpo, usabilidade e performance. Utilizando React para a construção da interface e Recharts para a criação de gráficos interativos.

Tecnologias utilizadas

  • React
  • Recharts
