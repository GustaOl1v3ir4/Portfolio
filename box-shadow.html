Gerar Box Shadow

O Gerar Box Shadow é uma ferramenta simples e intuitiva que permite aos usuários criar sombras personalizadas para elementos web. Com opções para ajustar deslocamento, desfoque, propagação e cor da sombra. Com esse projeto foi possível aprimorar minhas habilidades em JavaScript referente ao trabalho com Classes.

Tecnologias utilizadas

  • HTML
  • CSS
  • JavaScript
