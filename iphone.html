Clone Iphone 13

Está é uma réplica do site oficial do iPhone 13, recriada utilizando HTML e CSS e JavaScript para adicionar interatividade entre as opções de cores do produto. O projeto me ajudou a aprimorar minhas habilidades em design responsivo, layout com Flexbox e Grid, além de manipulação do DOM com JavaScript.

Tecnologias utilizadas

  • HTML
  • CSS
  • JavaScript
