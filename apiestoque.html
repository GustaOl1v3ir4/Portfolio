Api Estoque

O API ESTOQUE é uma aplicação backend desenvolvida para gerenciar o controle de produtos em estoque. O sistema permite cadastrar, listar, atualizar e remover produtos, garantindo a persistência dos dados em um banco MySQL. O projeto foi estruturado com Node.js e Express, e conta com autenticação JWT e integração com Docker para simplificar a execução e o ambiente de desenvolvimento.

Tecnologias utilizadas

  • Node.js
  • Express.js
  • MySQL
  • Sequelize (ORM)
  • Docker
  • JavaScript (ES6+)
